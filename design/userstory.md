# 가성비 지수 개발 서비스 - 유저스토리

## 마이크로서비스 구성
1. **Restaurant Search** - 식당 탐색 및 추천 서비스
2. **Order Management** - 주문 및 식사 경험 관리 서비스
3. **Value Assessment** - 가성비 평가 및 분석 서비스
4. **Personalization** - 개인화 학습 및 추천 서비스
5. **Admin Management** - 시스템 관리 및 운영 서비스

3. Value Assessment
1) 가성비 평가
UFR-VAL-010: [식사경험평가] 일반사용자로서 | 나는 나만의 가성비 기준을 만들기 위해 | 가격, 양, 맛, 서비스 4가지 요소를 상세하게 평가하고 싶다.
- 시나리오: 4요소 상세 평가
  식사 경험 완료 후 | 상세 평가를 시작하면 | 가격/양/맛/서비스 각 항목을 5점 척도로 평가할 수 있다.
  - [ ] 가격 평가: 예상 대비 실제 가격의 적정성
  - [ ] 양 평가: 개인 식사량 대비 충분함 정도
  - [ ] 맛 평가: 개인 취향 기준 만족도
  - [ ] 서비스 평가: 직원 친절도, 청결도, 분위기
  - [ ] 각 항목별 5점 척도 + 간단한 코멘트
  - [ ] 총 평가 시간 3분 이내 완료 가능
- M/5

  평가 완료 후 | 가성비 지수를 조회하면 | 개인 가중치가 반영된 점수와 동일 카테고리 평균 비교가 표시된다.
  - [ ] 개인 가중치 반영된 종합 가성비 점수 (100점 만점)
  - [ ] 가격/양/맛/서비스 항목별 기여도 표시
  - [ ] 동일 카테고리 식당들과의 비교 순위
  - [ ] 개인 기준 대비 높음/보통/낮음 표시
  - [ ] 가성비 계산 로직 간단 설명

  - [ ] 평가한 식당 목록과 각각의 가성비 점수
  - [ ] 개인 평가 패턴 분석 (엄격함/관대함 정도)
  - [ ] 선호 카테고리와 가성비 트렌드 그래프

  - [ ] 선호하는 음식 카테고리와 가격대

4. Personalization
1) 개인화 학습
UFR-PERS-010: [개인선호도학습] 학습시스템으로서 | 나는 사용자별 맞춤 추천을 위해 | 개인의 평가 패턴과 행동 데이터를 지속적으로 학습하고 싶다.

  사용자가 새로운 평가를 입력한 상황에서 | 시스템이 데이터를 분석하면 | 개인 선호도 모델이 자동으로 업데이트된다.

  - [ ] 기존 선호도 패턴과 비교하여 변화 감지
  - [ ] 최근 5회 평가가 기존 패턴과 50% 이상 차이 시 모델 재학습

  - [ ] 학습 완료 시 사용자에게 개인화 정확도 향상 알림

  - [ ] 개인 가성비 기준과 선호도 반영
  - [ ] 현재 위치, 시간대, 날씨 정보 고려
  - [ ] 최근 방문 이력으로 중복 방문 필터링
  - [ ] 예산 범위와 동행인 수 반영
  - [ ] 추천 이유와 예상 만족도 함께 제공
