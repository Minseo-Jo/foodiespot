# 고객 관찰 결과 보고서

## 관찰 목적
가성비가 좋고 대기 없이 식사하고 싶은 식당 이용자의 현재 행동 패턴, 인식, 감정 반응 및 불편 사항을 분석하여 서비스 기획을 위한 문제점과 잠재 니즈를 발굴

## 고객경험 단계
니즈발생 → 검색 → 평가 → 선택 → 이용 → 피드백

## 주요 발견 사항

### 1. 검색 채널의 다양성
- 네이버지도, 카카오맵, 배달앱, 인스타그램 등 다양한 플랫폼 사용
- 연령대별로 선호하는 검색 방식이 다름

### 2. 평가 기준의 차이
- 20대: 가격 우선, SNS 후기 중시
- 30대: 가격+품질 균형, 실용성 중시
- 40대: 양과 맛, 전통적인 평점 신뢰

### 3. 공통 불편사항
- 대기시간 예측 어려움
- 실제 가격과 예상 가격 차이
- 1인분 주문의 제약
- 실시간 운영 여부 확인 어려움

### 4. 잠재 니즈
- 실시간 대기시간 정보
- 가성비 기반 추천 시스템
- 1인 식사 친화적 옵션
- 시간대별 맞춤 추천
